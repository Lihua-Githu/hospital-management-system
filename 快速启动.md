# 🚀 快速启动指南

## ✅ 第一步：初始化数据库（只需执行一次）

### 方法1：使用Python脚本（推荐）
```powershell
python init_db.py
```
按提示输入MySQL的root密码，脚本会自动：
- 创建数据库和表
- 插入示例数据
- 自动修改app.py中的密码配置

### 方法2：使用批处理文件
双击运行 `init_database.bat`

### 方法3：手动执行
```powershell
"C:\Program Files\MySQL\MySQL Server 8.0\bin\mysql.exe" -u root -p < database\schema.sql
```

## ✅ 第二步：启动系统

### 方法1：使用批处理文件
双击运行 `start_server.bat`

### 方法2：使用命令行
```powershell
python app.py
```

## ✅ 第三步：访问系统

打开浏览器访问：http://localhost:5000

## ⚠️ 常见问题

### 问题1：端口5000被占用
修改 `app.py` 最后一行：
```python
app.run(debug=True, host='0.0.0.0', port=8080)
```

### 问题2：数据库连接失败
检查 `app.py` 第18-24行的配置是否正确

### 问题3：MySQL服务未运行
```powershell
Start-Service MySQL80
```
